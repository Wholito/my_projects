# Pisarik Guessr

Многопользовательская игра-угадайка на Flutter с использованием Firebase.  
Один игрок загадывает персонажа или предмет, другой угадывает по описаниям, составленным из слов на заданную букву.

## Описание

Приложение позволяет играть с друзьями в реальном времени.  
Поддерживаются две игровые вселенные: Brawl Stars и Dota 2.  
Для Dota 2 доступны два режима: угадывание героев или предметов.  
Включение Pussy Mode даёт подсказки: таблицу всех персонажей/предметов для угадывающего и список слов на букву для загадывающего.

## Особенности

- Мультиплеер в реальном времени (Firebase Firestore)
- Авторизация по email/паролю (Firebase Auth)
- Приглашение друзей по уникальному коду
- Загрузка аватарок через Cloudinary
- Тёмный адаптивный интерфейс (Material 3)
- Анимации, эффекты 

## Технологии

- Flutter (SDK >= 3.0)
- Firebase (Auth, Firestore)
- Provider (управление состоянием)
- Cloudinary (хранение изображений)
- Image Picker, Dio, Connectivity Plus
